# Bank Marketing Data Mining Project
This project applies supervised machine learning techniques to the UCI Bank Marketing dataset to predict whether a customer will subscribe to a term deposit.

Two classification models (Logistic Regression and Decision Tree) are developed and compared using multiple evaluation metrics including Accuracy, Precision, Recall, F1-score and ROC-AUC.

Dataset:

Bank Marketing Dataset

Source:
https://archive.ics.uci.edu/dataset/222/bank+marketing

Repository contents:

- Notebook
- Dataset
- Figures
- Final report
